# Quiz Box (PWA)

自分でクイズカードを作って遊べるクイズアプリのPWA版です。

## 使い方(公開・インストール手順)

このフォルダの中身をそのままどこかでホスティングしてください(**https経由が必須**です。ローカルの `file://` では Service Worker が動かず、オフライン対応やホーム画面追加がうまくいきません)。

手軽な方法の例:

1. **GitHub Pages**
   - GitHubで新しいリポジトリを作成
   - このフォルダの中身(`index.html`, `app.jsx`, `manifest.json`, `service-worker.js`, アイコン画像)をアップロード
   - Settings → Pages で公開
2. **Netlify / Vercel**
   - このフォルダをドラッグ&ドロップでデプロイ(会員登録は必要ですが無料枠で十分です)

## スマホでインストールする

1. 公開したURLをAndroidのChromeで開く
2. 右上のメニュー(⋮)→「アプリをインストール」または「ホーム画面に追加」をタップ
3. ホーム画面にアイコンが追加され、アプリのように起動できます(オフラインでも一度開いたページはキャッシュから表示されます)

## データについて

作成したクイズカードは、このアプリを開いている端末のブラウザ内(localStorage)に保存されます。他の端末やブラウザとは共有されません。

## 中身

- `index.html` — アプリの入り口(React/Babelを CDN から読み込みます)
- `app.jsx` — クイズアプリ本体のコード
- `manifest.json` — PWAとしてインストールするための設定
- `service-worker.js` — オフラインキャッシュ用
- `icon-192.png` / `icon-512.png` — アプリアイコン
